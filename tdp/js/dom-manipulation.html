Manipolazione del DOM — Esempi Didattici

Ogni sezione mostra il codice JavaScript usato e un'area interattiva per testarlo.

1. getElementById e modifica del testo

// Seleziona un elemento tramite id e ne modifica il contenuto testuale
const titolo = document.getElementById("titolo-demo");
titolo.textContent = "Testo modificato!";
titolo.style.color = "blue";

Questo è il testo originale.

2. querySelector e querySelectorAll

// querySelector → restituisce il PRIMO elemento che corrisponde al selettore CSS
const primo = document.querySelector(".frutto");
primo.style.background = "orange";

// querySelectorAll → restituisce una NodeList con TUTTI gli elementi corrispondenti
const tutti = document.querySelectorAll(".frutto");
tutti.forEach(function(el) {
    el.style.fontWeight = "bold";
});

3. classList — add, remove, toggle, contains

const box = document.getElementById("box-classe");

box.classList.add("evidenziato");      // aggiunge una classe
box.classList.remove("evidenziato");   // rimuove una classe
box.classList.toggle("rosso");         // aggiunge se non c'è, rimuove se c'è
box.classList.contains("grande");      // restituisce true / false

Questo paragrafo può ricevere e perdere classi CSS.

5. Navigare il DOM: parentNode, children, firstElementChild, lastElementChild, nextElementSibling

const el = document.getElementById("nodo-centrale");

el.parentNode                      // nodo genitore
el.parentNode.children             // tutti i figli del genitore (HTMLCollection)
el.parentNode.firstElementChild    // primo figlio elemento
el.parentNode.lastElementChild     // ultimo figlio elemento
el.nextElementSibling              // fratello successivo
el.previousElementSibling         // fratello precedente

6. Creare e inserire elementi: createElement, appendChild, insertBefore

const lista = document.getElementById("lista-dinamica");

// Crea un nuovo elemento
const nuovo = document.createElement("li");
nuovo.textContent = "Elemento aggiunto";
nuovo.classList.add("evidenziato");

// Aggiunge in fondo alla lista
lista.appendChild(nuovo);

// Inserisce PRIMA di un elemento esistente
const primoFiglio = lista.firstElementChild;
lista.insertBefore(nuovo, primoFiglio);

8. Attributi: getAttribute, setAttribute, removeAttribute

const link = document.getElementById("link-demo");

link.getAttribute("href");                        // legge l'attributo
link.setAttribute("href", "https://...");         // imposta / sovrascrive
link.setAttribute("target", "_blank");            // aggiunge un nuovo attributo
link.removeAttribute("target");                   // rimuove un attributo

9. addEventListener e removeEventListener

const box = document.getElementById("box-eventi");

// La funzione handler deve essere definita separatamente
// per poterla rimuovere in seguito con removeEventListener
function cambiaColore() {
    box.classList.toggle("evidenziato");
}

box.addEventListener("click", cambiaColore);    // aggiunge il listener
box.removeEventListener("click", cambiaColore); // rimuove lo stesso listener

// Altri eventi comuni: "mouseover", "mouseout", "keydown", "submit", "change"

10. innerHTML e textContent — differenze chiave

const box = document.getElementById("box-content");

// textContent → testo puro, i tag HTML non vengono interpretati (sicuro)
box.textContent = "<strong>Non viene interpretato</strong>";

// innerHTML → interpreta il markup HTML (potente, ma attenzione all'XSS)
box.innerHTML = "<strong>Viene interpretato</strong>";

11. Mostrare e nascondere elementi

const pannello = document.getElementById("pannello");

// Metodo 1: modifica diretta dello stile inline
pannello.style.display = "none";     // nasconde
pannello.style.display = "block";    // mostra

// Metodo 2: tramite classe CSS  (.nascosto { display: none; })
pannello.classList.add("nascosto");
pannello.classList.remove("nascosto");
pannello.classList.toggle("nascosto");
Questo è il pannello che può essere mostrato o nascosto.

12. Iterare una NodeList con forEach

// querySelectorAll restituisce una NodeList
// NodeList supporta forEach direttamente (a differenza di HTMLCollection)
const celle = document.querySelectorAll("#tabella-demo td");

celle.forEach(function(cella, indice) {
    cella.textContent = "Cella " + indice;
    if (indice % 2 === 0) {
        cella.classList.add("evidenziato");
    }
});
