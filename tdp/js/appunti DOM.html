Il DOM (Document Object Model)

Cos'è il DOM?

Il DOM (Document Object Model) è una rappresentazione ad albero della struttura di una pagina HTML. Quando il browser carica una pagina web, crea automaticamente un modello del documento che può essere manipolato tramite JavaScript.

Pensa al DOM come a un ponte tra il codice HTML statico e JavaScript dinamico: permette di leggere, modificare, aggiungere o eliminare elementi della pagina in tempo reale.

La Struttura ad Albero

Ogni elemento HTML diventa un nodo nell'albero del DOM. Ecco un esempio visivo:

Codice HTML: <!DOCTYPE html> <html> <head> <title>La mia pagina</title> </head> <body> <h1>Benvenuto</h1> <p>Questo è un paragrafo</p> </body> </html> Albero DOM corrispondente: document | <html> / \ <head> <body> | / \ <title> <h1> <p> | | | "La mia "Benvenuto" "Questo è pagina" un paragrafo"

Tipi di Nodi

  1. Element Node: gli elementi HTML (<div>, <p>, <h1>, ecc.)
  2. Text Node: il contenuto testuale dentro gli elementi
  3. Attribute Node: gli attributi degli elementi (class, id, src, ecc.)
  4. Document Node: la radice dell'albero (l'oggetto document)

Accedere agli Elementi

JavaScript offre diversi metodi per selezionare elementi nel DOM:

// Seleziona per ID let elemento = document.getElementById("mioId"); // Seleziona per classe (restituisce una collezione) let elementi = document.getElementsByClassName("miaClasse"); // Seleziona per tag (restituisce una collezione) let paragrafi = document.getElementsByTagName("p"); // Query selector (più moderno e flessibile) let primo = document.querySelector(".miaClasse"); // primo elemento let tutti = document.querySelectorAll(".miaClasse"); // tutti gli elementi

Manipolare il DOM

Una volta selezionato un elemento, puoi modificarlo in vari modi:

Modificare il Contenuto

// Cambiare il testo elemento.textContent = "Nuovo testo"; // Cambiare HTML interno elemento.innerHTML = "<strong>Testo in grassetto</strong>";

Modificare gli Attributi

// Leggere un attributo let valore = elemento.getAttribute("src"); // Impostare un attributo elemento.setAttribute("class", "nuovaClasse"); // Attributi comuni hanno proprietà dedicate elemento.id = "nuovoId"; elemento.className = "classe1 classe2";

Modificare lo Stile

// Cambiare proprietà CSS elemento.style.color = "red"; elemento.style.backgroundColor = "yellow"; elemento.style.fontSize = "20px"; // Aggiungere/rimuovere classi CSS elemento.classList.add("attiva"); elemento.classList.remove("nascosta"); elemento.classList.toggle("evidenziata");

Creare e Aggiungere Elementi

// Creare un nuovo elemento let nuovoDiv = document.createElement("div"); nuovoDiv.textContent = "Sono un nuovo div"; // Aggiungere l'elemento al DOM document.body.appendChild(nuovoDiv); // Inserire prima di un altro elemento let riferimento = document.getElementById("rifId"); document.body.insertBefore(nuovoDiv, riferimento);

Rimuovere Elementi

// Rimuovere un elemento elemento.remove(); // O dalla prospettiva del genitore let genitore = document.getElementById("contenitore"); let figlio = document.getElementById("daCancellare"); genitore.removeChild(figlio);

Navigare nel DOM

Puoi spostarti tra i nodi usando le proprietà di relazione:

let elemento = document.getElementById("mioId"); // Nodi genitore e figli elemento.parentNode; // nodo genitore elemento.childNodes; // tutti i nodi figli elemento.firstChild; // primo figlio elemento.lastChild; // ultimo figlio // Solo elementi (ignora nodi di testo) elemento.parentElement; // elemento genitore elemento.children; // elementi figli elemento.firstElementChild; // primo elemento figlio elemento.lastElementChild; // ultimo elemento figlio // Fratelli (siblings) elemento.nextSibling; // nodo successivo elemento.previousSibling; // nodo precedente elemento.nextElementSibling; // elemento successivo elemento.previousElementSibling; // elemento precedente

Eventi e Interazione

Il DOM permette di rispondere alle azioni dell'utente tramite gli eventi:

// Aggiungere un event listener let bottone = document.getElementById("mioBottone"); bottone.addEventListener("click", function() { alert("Bottone cliccato!"); }); // Eventi comuni: // - click: clic del mouse // - mouseover: mouse sopra l'elemento // - mouseout: mouse esce dall'elemento // - keydown: tasto premuto // - keyup: tasto rilasciato // - submit: invio di un form // - change: cambio di valore in input // - load: caricamento completato

Esempio Pratico Completo

<!DOCTYPE html> <html> <head> <title>Esempio DOM</title> <style> .evidenziato { background-color: yellow; } </style> </head> <body> <h1 id="titolo">Lista della Spesa</h1> <ul id="lista"> <li>Pane</li> <li>Latte</li> </ul> <input type="text" id="nuovoItem" placeholder="Nuovo elemento"> <button id="aggiungi">Aggiungi</button> <script> // Seleziona elementi let bottone = document.getElementById("aggiungi"); let input = document.getElementById("nuovoItem"); let lista = document.getElementById("lista"); // Aggiungi evento al bottone bottone.addEventListener("click", function() { // Leggi il valore dell'input let testo = input.value; if (testo !== "") { // Crea nuovo elemento li let nuovoLi = document.createElement("li"); nuovoLi.textContent = testo; // Aggiungi alla lista lista.appendChild(nuovoLi); // Pulisci l'input input.value = ""; } }); // Evidenzia il titolo al passaggio del mouse let titolo = document.getElementById("titolo"); titolo.addEventListener("mouseover", function() { titolo.classList.add("evidenziato"); }); titolo.addEventListener("mouseout", function() { titolo.classList.remove("evidenziato"); }); </script> </body> </html>

Schema Riassuntivo: Flusso di Lavoro con il DOM

1. SELEZIONA ↓ document.getElementById() document.querySelector() 2. MANIPOLA ↓ .textContent = "..." .innerHTML = "..." .style.color = "..." .classList.add("...") 3. CREA/AGGIUNGI ↓ document.createElement() .appendChild() 4. ASCOLTA EVENTI ↓ .addEventListener("click", function)

Punti Chiave da Ricordare

  • Il DOM è una rappresentazione ad albero del documento HTML
  • document è l'oggetto principale per accedere al DOM
  • Gli elementi possono essere selezionati, modificati, creati e rimossi
  • JavaScript rende le pagine web dinamiche e interattive tramite il DOM
  • Gli eventi permettono di reagire alle azioni dell'utente
  • Ogni modifica al DOM si riflette immediatamente nella pagina visualizzata
Consiglio didattico: Fai provare agli studenti esempi pratici, modificando elementi in tempo reale nella console del browser (F12) per vedere immediatamente i risultati!
