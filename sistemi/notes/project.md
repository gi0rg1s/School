# Lybrary Management System
## Project Overview

The Lybrary Management System is a program that wants to simulate a virtual Lybrary where users can manage books.
## Features
- Add new books to the Lybrary
- Remove books from the Lybrary
- Modify book details
- Search for books by title, author, or genre
- View a list of all books in the Lybrary (even ordinated by different criteria)
- Borrow and return books
- User authentication and management
- Show books statistics (most borrowed books, most voted books, etc.)

## User Interface
- Wishlist
- Read books
- Currently reading books
- Book ratings and reviews
- Reading challenges and goals
- Personalized statistics and achievements

## Features to be considered for future versions
- Social features (friend lists, book recommendations, etc.)
- Integration with external book databases (Google Books API, etc...)

## Technologies Used
- Programming Language: C
- Graphical User Interface: SDL2 or GTK
